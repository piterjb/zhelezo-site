Personal Portfolio Website
Welcome to my personal portfolio website repository! This modern, responsive portfolio is built with Next.js 14, showcasing my skills, projects, and professional journey.
About This Site
This portfolio website serves as a comprehensive showcase of my work and expertise. Built with the latest web technologies, it features:

Professional Projects: Detailed case studies of my most significant work
Skills Overview: Visual representation of my technical abilities
Experience Timeline: My professional journey and accomplishments
Blog Section: Thoughts and insights on technology and development
Contact Form: An easy way for potential clients and employers to reach me

Tech Stack
This portfolio is built with modern technologies for optimal performance and user experience:

Next.js 14: Server-side rendering, fast page loads, and optimized routing
React 19: Component-based UI architecture
Tailwind CSS & shadcn UI: Responsive design and modern UI components
Google Forms Integration: Secure contact form processing
Analytics: Performance tracking via Google Analytics and Vercel Analytics
SEO Optimized: Structured for maximum search engine visibility

Performance
The site achieves a perfect 100% performance score on Lighthouse, ensuring visitors have a seamless experience regardless of their device or connection speed.

Local Development
To run this project locally:

Clone this repository
Copy the contents of .env.copy to a new .env file and add your environment variables
Install dependencies:
bashnpm install

Start the development server:
bashnpm run dev

Visit http://localhost:3000 in your browser
